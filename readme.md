
# Loka.js
#### The first (un)official loka api!


Welcome to the loka.js api by itsredstonepro#0979

#### Installation:
`npm i loka.js axios`


more docs coming soon

#### Api Tree

- territory
    - get
        - byId
        - byNumb
        - byWorldAndNumb
    - custom
        - byTown
- town
    - get
        - byName
        - byArea
        - byTown
    - custom
        - userTownSearch (for user interaction)
- alliance
    - get
        - byName
        - fullList
- player
    - get
        - byName
        - byUuid
        - fullList
- market_buy
    - get
        - byType
        - fullList
- market_sales
    - get
        - byType
        - fullList
- battle
    - get
        - battles
- transaction
    - get
        - byName
        - byType
        - byOwnerUuid
        - byBuyerUuid
